# Diags76

Diags76 est un projet visant à fournir une plateforme web pour la gestion et la visualisation des diagnostics immobiliers. Ce projet est développé en utilisant PHP et MySQL, et est hébergé sur un serveur XAMPP.

## Fonctionnalités

- **Gestion des diagnostics** : Ajout, modification et suppression des diagnostics immobiliers.
- **Visualisation des données** : Affichage des diagnostics sous forme de tableaux et graphiques.
- **Authentification** : Système de connexion pour les utilisateurs autorisés.
- **Interface utilisateur** : Interface web intuitive et facile à utiliser.

## Installation

1. Clonez le dépôt GitHub :
    ```bash
    git clone https://github.com/votre-utilisateur/diags76.git
    ```

2. Déplacez-vous dans le répertoire du projet :
    ```bash
    cd diags76
    ```

3. Configurez votre serveur XAMPP pour pointer vers le répertoire du projet.

4. Importez la base de données MySQL en utilisant le fichier `database.sql` fourni.

5. Modifiez le fichier `config.php` pour y ajouter vos informations de connexion à la base de données.

## Utilisation

1. Démarrez votre serveur XAMPP.
2. Accédez à l'application via votre navigateur web à l'adresse `http://localhost/diags76`.
3. Connectez-vous avec vos identifiants pour commencer à utiliser la plateforme.
